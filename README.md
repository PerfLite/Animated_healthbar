# Animated Health Bar Neo

A NeoForge mod for Minecraft 1.21.1 that replaces the vanilla health and hunger bars with smooth animated custom bars.

## Features

- **Smooth animation** — health and hunger bars animate smoothly when values change
- **Effect-based textures** — the health bar changes color depending on active effects:
  - ❤️ Red — normal
  - 🟣 Purple — Wither
  - 🟢 Green — Poison
  - 🔵 Blue — Freeze
  - 🔥 Orange — Fire
  - 💚 Lime — Regeneration
  - 💛 Yellow — Absorption
- **Animated dots** — subtle dot animation on the bar for a polished feel
- **Hidden in Creative/Spectator** — bars are automatically hidden when not in survival mode
- **Resource pack friendly** — all textures are easily replaceable

## Installation

1. Install [NeoForge 1.21.1](https://neoforged.net/)
2. Download the mod `.jar` file
3. Place it in your `mods` folder
4. Launch the game!
